# Flyingwood Furniture Project Report

## Project Overview
FlyingWood Furniture is a comprehensive, modern e-commerce platform dedicated to offering premium wooden and sustainable furniture. The system is built using the MERN stack (MongoDB, Express.js, React.js, Node.js) and aims to provide an advanced shopping experience with a focus on immersive 3D product previews and seamless functionality.

## Core Features

### 1. Authentication & User Management
- Secure user registration and login using JWT (JSON Web Tokens) and bcryptjs for password hashing.
- Role-based access control supporting multiple roles (Customer, Vendor, Admin).
- User profile management.

### 2. Immersive 3D Product Experience
- Integrated 3D model support allowing users to view products from all angles before purchasing.
- Support for complex multi-file 3D models (GLTF/GLB formats) uploaded and served seamlessly.
- Dedicated 3D Showroom component for premium interactive experience.

### 3. Dynamic Product Catalog
- Advanced filtering options (by category, materials, price range).
- Product sorting (newest, price ascending/descending, popularity).
- Pagination system to handle large sets of products efficiently.
- View count tracking for displaying 'popular' or 'trending' items.

### 4. Vendor & Admin Capabilities
- **Product Management:** Vendors and admins can create, update, and delete products. Cloudinary is used for uploading and managing standard product images, alongside the 3D model files.
- **Admin Dashboard:** Access to site-wide statistics, including total users, revenue, pending orders, and low-stock warnings.
- User management and role assignment for admins.

### 5. Shopping Cart & Order Processing
- Persistent shopping cart supporting adding, removing, clearing, and quantity limits based on actual stock.
- Order creation that accurately calculates subtotals, taxes, and shipping integrations.
- Support for Stripe integration to securely process online payments.

### 6. Technical Stack
- **Frontend:** React, React Router
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (using Mongoose schemas)
- **File Management:** Multer & Cloudinary
- **Security & Extras:** dotenv config, CORS, Nodemailer for transactional emails

## Status
The project is set up for local development (with back-end and front-end simultaneously) and supports cloud deployment options.
